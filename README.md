# nextPage
支持快捷键方式对网站进行翻页（非滚动翻页）
# 1、安装方式
## 1）方式一：代码安装
- 步骤一：下载最新代码，并解压缩至本地某个目录
- 步骤二：打开chrome浏览器，在地址栏中输入chrome://extensions进入插件管理界面
- 步骤三：chrome右上角激活“开发者模式”
- 步骤四：选择左上角“加载已解压缩的扩展程序”，选择步骤一解压的代码目录
## 2）方式二：安装包安装（简单，但有可能会失败）
- 步骤一：下载最新安装包 [nextPage.crx](https://github.com/DogRex/nextPage/releases/download/v1.0/nextPage.crx)
- 步骤二：将安装包拖入chrome浏览器
# 2、使用方式
- 翻往下一页：组合快捷键 ctrl + ]
- 翻往上一页：组合快捷键 ctrl + [
# 3、说明
目前只识别当前页面中带有'下一页'和'上一页'的超链接，可能会存在奇怪的问题，请不要惊慌。
